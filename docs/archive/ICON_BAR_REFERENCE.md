## Color Scheme

- **Primary Color:** Orange/Primary-600
- **Hover State:** Primary-50 background with Primary-600 text
- **Icons:** Gray-600 default, Primary-600 on hover
- **Popover Background:** White with subtle shadow
- **Text:** Gray-900 for headings, Gray-500 for descriptions

## Interaction States

1. **Default:** Gray icons with subtle hover effect
2. **Hover:** Background changes to light primary color
3. **Active/Open:** Icon highlighted in primary color
4. **Focus:** Ring outline for accessibility

- вң… Keyboard navigation support
- вң… ARIA labels for screen readers
- вң… Focus indicators
- вң… Proper heading hierarchy
- вң… Touch-friendly sizes (minimum 44px)
- вң… High contrast text
