# Product Overview

Ghumo Phiro India is a tour booking platform specializing in custom tours across India, with a focus on Rajasthan and the Golden Triangle region.

## Core Features

- Public tour browsing and booking
- Custom tour request system
- User account management with booking history
- Admin dashboard for managing tours, bookings, inquiries, and custom requests
- JWT-based authentication with role-based access control (admin/user)

## User Roles

- **Public Users**: Browse tours, make bookings, submit inquiries and custom tour requests
- **Registered Users**: Access booking history and account management
- **Admin Users**: Full access to dashboard, manage all tours, bookings, inquiries, and custom requests

## Key Entities

- Tours (with itineraries, pricing, images, highlights)
- Bookings (with payment and status tracking)
- Inquiries (contact form submissions)
- Custom Tour Requests (personalized tour planning)
- Destinations (popular locations)
- Users (authentication and profiles)

## Business Model

Tour operator platform connecting travelers with curated Indian tours, offering both pre-packaged tours and custom itinerary planning services.
